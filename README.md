# Bulletproof BT

Event-driven, bias-resistant backtest engine scaffold.

## Install

```bash
python -m venv .venv && source .venv/bin/activate
pip install -e .
```

## Run (placeholder)

```bash
python scripts/run_backtest.py --config configs/engine.yaml
```

## Notes

`data/` and `outputs/` are runtime folders and are gitignored. Scripts will create
and manage them later.
